This package contains the implementation of the image interpolation algorithm presented in the paper:


"Efficient Image Interpolation via Content-Aware Clustering Fusion", Zongliang Gan, Kaihua Zhang.

This package contains a set of Matlab files that have been tested on 
Matlab R2016b  on Win 10 64

test_ip.m ---  Evaluation on the simulated image datasets
test_offline_opt.m -- Evaluation on offline C++
test_realworld_ip --- Evaluation on the real-world image dataset
